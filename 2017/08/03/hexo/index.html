hexo常用命令

hexo部署

hexo clean : 清除。需要重新部署时（如修改主题之后），需要运行此命令
hexo g : 生成
hexo d : 发布
hexo d -g : 上两个命令的合成

写文章

发布文章有两种方式

hexo new "title" : 新建一篇博客
hexo new draft "title" : 新建草稿
hexo publish "title" : 将草稿发布
