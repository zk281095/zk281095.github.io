sort问题

es会将string型数据检索时默认analysis，因此sort字符串时需要修改mapping中的字段。可以对field建两种方式，需要sort时sort(id.raw)

"id":{
    "type":"string",
    "fields":{
        "raw": {
            "type":"string",
            "index":"not_analyzed"
        }
    }
}
